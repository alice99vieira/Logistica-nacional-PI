Previsão de Consumo de Diesel no Brasil

Projeto desenvolvido para a disciplina Processamento da Informação da UFABC (Universidade Federal do ABC).

Descrição

Este projeto em Python utiliza dados históricos de vendas de óleo diesel no Brasil para estimar o consumo futuro utilizando regressão linear simples.

O programa lê um arquivo CSV contendo informações de vendas de diesel por município, filtra os dados de um estado escolhido pelo usuário e calcula o consumo total por ano. A partir desses dados históricos, é criado um modelo matemático que permite prever o consumo para os próximos anos.

Objetivo

Simular um cenário de planejamento logístico de combustível, onde é necessário prever a demanda antes que ela aconteça, auxiliando na tomada de decisões sobre produção e distribuição.

Tecnologias utilizadas

Python

Leitura de arquivos CSV

Estruturas básicas de programação

Regressão linear simples

Fonte dos dados

Dados públicos da ANP – Agência Nacional do Petróleo, Gás Natural e Biocombustíveis.

Como executar

Execute o arquivo Python no terminal:

python previsao_diesel.py

Depois informe a sigla do estado (UF) quando solicitado.
